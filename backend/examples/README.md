# Example lab

`example-lab.md` is a single-file lab archive that demonstrates every guide
feature: Markdown, highlighted code/YAML, hover copy buttons, per-user
credentials, and images.

## Use it

Admin → **Lab management** → **Import** → choose `example-lab.md`. The lab appears
as `example-lab`; open it to see the rendered result, and read the source file as
a copy-paste template for authoring your own labs.

Images must be **absolute URLs** (e.g. a public S3 object) so they survive
export/import — see the note in `04-images.md`.
